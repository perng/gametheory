Websocket Chatroom About

About

Websocket Chatroom is an online chatroom for people to communicate and socialize.

With a small but powerful set of console commands provided, people can freely create new chatrooms (and thus become room master), set it as public or private, set its max size, join and invite people to join. Multiple login is allowed. People can see what rooms are online, all the users online and who are in their current chatroom. They can change personal settings in the settings panel, including sound and background image options.

Besides chatting people can also play games. At this time one game (6-piece chess) is available.

Privacy

People are suggested not to disclose any sensitive personal information online.

Features to come

In the long run, these features could be included:

  • Change font style, color and size (server side support already finished).
  • Share image (server side support already finished).
  • The 6-piece chess game right now has human/computer and computer/computer play modes. Remote human/human play mode will finish development soon.
  • More online games can be added.
  • Record chat message. This is optional at this time since this is not a commercial application.
  • Add a live video window.
  • Add a chatbot.
  • Other features. Recommendations are welcome.

Technology

Websocket Chatroom is created based on the websocket technology. Front end is written in HTML5/Javascript/CSS. Back end websocket server is written in Python|Autobahn.
